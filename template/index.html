JMZ Design Program

Comprehensive Stator Design Software
May 2017 - May 2018

Lead programmer for Toshiba's comprehensive low and medium voltage (up to 6 kV) stator design program. The JMZ design program is used as the backbone for all custom stator designs that go through Toshiba’s Houston Motor Plant.

The JMZ design program is used as the backbone for all custom stator designs that go through Toshiba’s Houston Motor Plant

The JMZ design program completely standardizes the motor design process. The program also introduces the use of a comprehensive motor design database. Using inputs from the designer, the program pulls associated data from the database and applies relevant mathematical formulas to completely generate a customized bill of material set that uploads and configures into Oracle’s EBS application. The program also auto-generates standardized shop floor-ready custom design output sheets upon successful run.

The introduction of the program has more than halved the normal custom design time required by engineers and has eliminated many sources of human error.

The program has received multiple in-company awards related to quality and innovation and continues to be used at TIC.

Due to the proprietary nature of company software, the code and live demo features are unavailable for viewing.

Technologies Employed
Visual Basic & Oracle's powerful scripting software, Dataload with PHP
